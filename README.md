# meflix

Base do MVP para transformar a vida do usuário em uma série de streaming, agora com foco mobile.

## Estrutura atual

- `backend`: API Express com Gemini, Supabase, PIX e webhook
- `mobile`: app Expo com quiz, preview, checkout e resultado
- `frontend`: base web antiga, mantida por enquanto mas fora do foco principal

## Fluxo do produto

1. Quiz
2. Geração da prévia
3. Checkout PIX
4. Liberação do resultado
5. Compartilhamento
